# azure-static-website-project
My first beginner cloud project deploying a static website using microsoft Azure 
https://firstbeginnerproj.z33.web.core.windows.net/

I built:
- a static HTML website
- Hosted on Azure storage
- Publicly accessible via Azure endpoint URL

Azure services used:
- Azure Storage Account
- Static Website Hosting feature
- Blob Storage container ($web)

What I learned
- Basics of cloud computing and hosting
- How static website hosting works in Azure
- How to deploy and update files in the cloud
- Understanding of public endpoints and DNS concepts


This project was built as part of my cloud engineering learning path and serves as a foundation for more advanced projects using Linux, networking, and Terraform.
